# シフト表印刷ツール

## 概要
社員名簿から毎月のシフト表を生成し、A4横1ページ（2段階構成）で印刷できるツール。

## 目的
シフト表作成を自動化することで、毎月のシフト表作成コストを削減する。

## 特徴
- 月初めローテーション切り替え方式
- UTF基準により日数差分を計算
- 簡易エディタによる data.js の生成
- 静的ファイル3種（index.html / bundle.js / data.js）で配布可能

## 技術スタック
- React.js
- Typescript
- Tailwind CSS
- esbuild
